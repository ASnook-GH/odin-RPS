# odin-RPS

## Odin Project - Rock Paper Scissors

Project to recreate the game Rock Paper Scissors in Javascript.
the game is to be played entirely in the console. no GUI is needed yet.

### Suggested Approach

1. Plan / Pseudo-code the solution
2. Write the code
3. Test the code
4. Iterate

## Thoughts on the project

the initial set-up of the went well & quickly : getting the computer choice & player choice didn't take too much time.
finding the comparaison logic and coding it was easy but I'm certain there's a better way of writing it. With 3 scenarios, it wasn't too long but if there were more, my approach would have been far too time consuming.

the biggest difficulty was getting the rounds to "loop" and retain the information with each new round. this was likely to me have a poor understanding of the syntax and not using the correct operators. (for example, at line 83 using "round = 13" for a long time before understanding it needs to be "round === 13")

all in all, I feel happy about how I approach the project and worked on it. thinking about it with pseudocode made it easier to approach and break into smaller parts.
